    Editing a VCEM server

    Using the Discovering VCEM server Management application, you can edit some VCEM server fields.

    To edit the VCEM server discovery information, complete the following steps.
    1. Select Discover > VCEM Server.
      The Discover VCEM Servers dialog box displays.
    2. Select a VCEM server in the Discovered VCEM Servers dialog box and click Edit.
      The Edit VCEM Server dialog box displays.
    3. Change the IPv4 address or fully qualified domain name in the Network Address field.
    4. Change the VCEM server SOAP API port number in the Port field.
    5. Enter the VCEM server username in the User ID field.
    6. Enter the VCEM server user password in the Password field.
    7. Click OK on the Edit VCEM Server dialog box.
    8. Click Close on the Discover VCEM Servers dialog box.
