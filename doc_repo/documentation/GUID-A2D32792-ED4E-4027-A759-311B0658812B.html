    Troubleshooting VCEM server active discovery

    Problems with active discovery can occur due to setup issues.

    If you encounter discovery problems, complete the following checklist to ensure that discovery setup is proper.

    1. Verify IP connectivity by opening the command prompt and entering a ping command to the device IP address. 
      ping <Device_IP_Address>
    2. Verify that the HP VCEM server is running. Enter the IP address with the SOAP API port number, using HTTPS,
      for example, https://10.1.1.11:50001.
    3. Verify that the user ID and password are correct for the VCEM server.
