    Deleting a VCEM server from discovery

    A VCEM server can be deleted permanently from active discovery.

    1. Select Discover > VCEM Server.
      The Discover VCEM Servers dialog box displays.
    2. Select one or more VCEM servers to be deleted from active discovery in the Previously Discovered Addresses table.
    3. Click Delete.
      The specified VCEM servers are deleted permanently from active discovery.
    4. Click Close on the Discover VCEM Servers dialog box.
