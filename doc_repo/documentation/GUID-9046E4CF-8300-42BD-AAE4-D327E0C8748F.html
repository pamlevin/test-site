    VCEM integration support in SMI-A

    The Brocade Storage Management Initiative Agent (SMI-A) supports the discovery of Virtual Connect Enterprise Manager (VCEM) servers.

    NOTE
    Discovery of TopologyView and SANActiveConnections will be incomplete for VC modules connected to the end devices.

    HP SAN Network Advisor supports Virtual Connect discovery for the HP Virtual Connect 8Gb 24-port Fibre Channel module and Virtual Connect 8Gb 20-port Fibre Channel module through the HP Virtual Connect Enterprise Manager (VCEM) server.

    The AG profile and NPIV profile is updated to support the VirtualConnect modules of VCEM integration:

    • Instances of Brocade_AccessGateway (VC module) for ElementConformsToProfile are not returned as a complete topology. Therefore Brocade_AccessGateway of the VC module does not conform to the AG and NPIV profile.
    • Instances of the following classes will not be returned for Virtual Connect because external management access points are disabled.
      • Brocade_ElementConformsToProfile
      • Brocade_AGWebGUIRemoteAccessPoint
      • Brocade_AGMgmtAccessPoint
      • Brocade_AGHostedRemoteAccessPoint
      • Brocade_AGHostedMgmtAccessPoint
    • The Brocade_AGSoftwareIdentity.Manufacturer value is populated as “HP” for VC modules.
