    Returning a VCEM server to active discovery

    After a VCEM server has been removed from the active discovery list, it can be returned to active discovery.

    Use the following steps to return a VCEM server to active discovery to allow the Management application to discover and monitor the VCEM server.

    1. Select Discover > VCEM Server.
      The Discover VCEM Servers dialog box displays.
    2. Select the VCEM server that you want to return to active discovery in the Previously Discovered Addresses table.
    3. Click Discover.
      The recovered VCEM server displays in the Discovered VCEM Servers table.
    4. Click Close on the Discover VCEM Servers dialog box.
