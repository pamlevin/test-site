    Virtual Connect Enterprise Manager Server Discovery

    • VCEM overview
      The Management application enables you to discover Virtual Connect Enterprise Manager (VCEM) servers.
    • Discovering a VCEM server
      A Virtual Connect Enterprise Manager (VCEM) server can be discovered using the Management application.
    • Editing a VCEM server
      Using the Discovering VCEM server Management application, you can edit some VCEM server fields.
    • Removing a VCEM server from active discovery
      A VCEM server can be removed from the active discovery list of the VCEM server Management application.
    • Returning a VCEM server to active discovery
      After a VCEM server has been removed from the active discovery list, it can be returned to active discovery.
    • Deleting a VCEM server from discovery
      A VCEM server can be deleted permanently from active discovery.
    • Viewing the VCEM server discovery state
      VCEM server discovery status can be viewed through the Management application.
    • Troubleshooting VCEM server active discovery
      Problems with active discovery can occur due to setup issues.
    • Virtual Connect module product icons
    • Virtual Connect module right-click menu
      Information about a Virtual Connect module is accessible from the right-click menu.
    • Viewing Virtual Connect module properties
      Virtual Connect module properties are displayed on the Properties menu option.
    • VCEM integration support in SMI-A
      The Brocade Storage Management Initiative Agent (SMI-A) supports the discovery of Virtual Connect Enterprise Manager (VCEM) servers.
