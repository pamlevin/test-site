Configure Email Server
Configure the SMTP mail server to send email notifications to all users enabled to receive them. This is a convenient way to keep track of events on the SAN networks.
1.
Click the Settings icon ( ), and then select Network Monitoring > Email Setup.
Figure 12. Email setup page
2.
Enter the configuration information for the SMTP email server.
3.
Select the Enable checkbox.
4.
Click Save.
